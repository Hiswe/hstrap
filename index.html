Overview

Hstrap is a little lib build on top of Stylus.
The main purpose of hstrap is to give some handy and/or uncommon mixins.

It provides also some particular design classes for buttons, forms & other things.

It's not (and never will) be a complete framework as twitter bootstrap, fluidity or roots.
For reset, typography, grid, tables and so on, look at them.

Install

This project can be found on github

npm install hstrap
Use it with Stylus

bower install hstrap
Only CSS files
